A HTTP/1.1 server built from scratch in C — no libraries, no framework. Handles socket management, raw request parsing, response generation, static file serving, and CGI script execution. Written for macOS using POSIX APIs and compiled with clang.

HTTP/1.1 GET · HEAD · OPTIONS · POST Static file serving CGI support Keep-Alive connection FSM request parser Configurable via config.h
Architecture 
src/
├── lib/
│   ├── http/   ← protocol implementation
│   └── net/    ← socket & connection handling
├── config.h    ← port, backlog, request limits
└── main.c      ← entry point
